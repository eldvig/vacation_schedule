# 📅 График отпусков и дежурств сотрудников

Интерактивная HTML-страница для визуализации графика отпусков и дежурств сотрудников в виде диаграммы Ганта. Поддерживает загрузку данных из Excel, фильтрацию по отделам, отображение пересечений отпусков и множество других функций.

## ✨ Возможности

### Навигация и управление
- ✅ Переключение масштаба: Дни / Недели / Месяцы
- ✅ Кнопки навигации: ◀◀ В начало | ◀ Назад | Вперёд ▶ | ▶▶ В конец

- ✅ Активные состояния кнопок (подсветка включённых режимов)
- ✅ Тултипы при наведении на отпуска, дежурства и кнопки

### Данные
- ✅ Сохранение порядка сотрудников в localStorage

## 🚀 Быстрый старт

### Вариант 1: Открыть готовый файл

## 📊 Формат данных

### Встроенные данные (rawData / rawDuty)

Формат: `Имя сотрудника` \t `ДД.ММ.ГГГГ` \t `количество дней` \t `отдел (опционально)`

```
Сотрудник_1	15.01.2026	10	Отдел А
Сотрудник_1	01.03.2026	5	Отдел А
Сотрудник_2	01.01.2026	14	Отдел Б
Сотрудник_3	01.01.2026	7	
```

> **Примечание:** Если отдел не указан — поле может быть пустым. Такие сотрудники попадут в категорию "(без отдела)".

## 📁 Структура Excel файла

Excel файл должен содержать **два листа**:

### Лист "отпуска"
| A (Сотрудник) | B (Дата начала) | C (Дней) | D (Отдел) |
|---|---|---|---|
| *Заголовки* | *любые* | *любые* | *любые* |
| Иванов И.И. | 15.01.2026 | 14 | Отдел разработки |
| Петров П.П. | 20.01.2026 | 10 | Отдел тестирования |

### Лист "дежурства"
| A (Сотрудник) | B (Дата начала) | C (Дней) | D (Отдел) |
|---|---|---|---|
| *Заголовки* | *любые* | *любые* | *любые* |
| Иванов И.И. | 25.01.2026 | 1 | Отдел разработки |

### Поддерживаемые названия листов
- **Отпуска:** `отпуска`, `Отпуска`, `ОТПУСКА`, `vacations`, `Vacations`
- **Дежурства:** `дежурства`, `Дежурства`, `ДЕЖУРСТВА`, `duties`, `Duties`

Поиск регистронезависимый — можно использовать любое написание.

### Поддерживаемые форматы дат
- ✅ Текстовый формат: `15.01.2026`, `01.03.2026`
- ✅ Excel-формат даты (числовой serial date)
- ✅ JavaScript Date объекты

> **Важно:** Столбец D (Отдел) — **необязательный**. Если его нет, все сотрудники будут отнесены к категории "(без отдела)".

## 📖 Руководство пользователя

### Панель управления

| Кнопка | Описание |
|---|---|
| 📥 **Загрузить Excel** | Загрузка данных из Excel-файла |
| 🎲 **Цвета** | Перегенерация случайных цветов для сотрудников |
| 👁️ **Пересечения** | Показать/скрыть красные зоны пересечений отпусков |
| 👁️ **Сегодня** | Показать/скрыть линию текущего дня + центрировать график |
| 👁️ **Дежурства** | Показать/скрыть чёрные полосы дежурств |
| **Дни / Недели / Месяцы** | Переключение масштаба отображения |
| ◀◀ | Прокрутка в самое начало |
| ◀ | Прокрутка влево |
| ▶ | Прокрутка вправо |
| ▶▶ | Прокрутка в самый конец |

### Drag-and-drop
- Наведите курсор на имя сотрудника в левой колонке
- Зажмите левую кнопку мыши и перетащите строку вверх или вниз
- Отпустите кнопку мыши над нужной позицией
- Порядок автоматически сохранится в браузере

### Фильтрация по отделам
1. Прокрутите страницу ниже графика до блока **"🏢 Фильтр по отделам"**
2. Снимите галочки с ненужных отделов
3. График автоматически перерисуется с учётом фильтра
4. Используйте кнопки **"Выбрать все"** / **"Снять все"** для быстрого управления

### Масштаб отображения
- **Дни** — каждая ячейка = 1 день (20px)
- **Недели** — каждая ячейка = 7 дней (~40px)
- **Месяцы** — каждая ячейка = месяц (~120px)

При переключении масштаба график автоматически центрируется на сегодняшнем дне.

## ⚠️ Известные особенности

### Часовые пояса
Приложение корректно работает в любом часовом поясе. Все даты обрабатываются в локальном времени, кроме расчёта разницы в днях (используется `Date.UTC` для защиты от перехода на летнее время).

### Совместимость с браузерами
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Edge 90+
- ✅ Safari 14+

### Ограничения
- Для загрузки Excel требуется интернет (для подгрузки SheetJS с CDN)
- Drag-and-drop не работает на сенсорных устройствах
- Порядок сотрудников сохраняется только для данных из `rawData` (при загрузке Excel сбрасывается)

### localStorage
Приложение использует `localStorage` для сохранения:
- Порядка сотрудников (`employeeOrder`)

Данные хранятся только локально в браузере и не передаются на сервер.

## 🎨 Кастомизация

### Изменение демо-данных
Откройте HTML-файл в текстовом редакторе и найдите константы в начале `<script>`:

```javascript
let rawData = `Сотрудник_1	15.01.2026	10	Отдел А
...`;

let rawDuty = `Сотрудник_1	20.01.2026	3	Отдел А
...`;
```

Замените содержимое на свои данные, соблюдая формат.

### Изменение стилей
Все стили находятся в блоке `<style>` в начале файла. Ключевые классы:
- `.vacation-bar` — полосы отпусков
- `.duty-bar` — полосы дежурств
- `.overlap-zone` — зоны пересечений
- `.today-line` — линия "Сегодня"
