📅 График отпусков и дежурств сотрудников

📥 Загрузить Excel
🚀 Возможности и фичи:
  • ✅ Диаграмма Ганта с отпусками и дежурствами сотрудников
  • ✅ Дежурства отображаются полупрозрачным чёрным цветом с белой границей поверх отпусков
  • ✅ Отображение отпуска больше по вертикали, чем дежурство (42px vs 22px)
  • ✅ Кнопка скрытия/показа дежурств
  • ✅ Sticky шапка — при вертикальной прокрутке заголовок с календарём остаётся на месте
  • ✅ Sticky колонка "Сотрудник" — при горизонтальной прокрутке имена сотрудников видны слева
  • ✅ Пересечения отпусков с правильной логикой разделения по составу сотрудников
  • ✅ Drag-and-drop сотрудников (порядок сохраняется в localStorage при использовании данных из rawData; при загрузке из Excel сбрасывается)
  • ✅ Случайные цвета при каждом запуске + кнопка 🎲 для перемешивания
  • ✅ Переключение масштаба: Дни / Недели / Месяцы
  • ✅ Кнопки навигации: Назад / Вперёд / В начало / В конец
  • ✅ Линия "Сегодня" с возможностью скрытия
  • ✅ Центрирование графика на сегодняшнем дне при нажатии "Сегодня", "Дни", "Недели", "Месяцы"
  • ✅ Тултипы при наведении на отпуск или дежурство
  • ✅ Статистика внизу страницы
  • ✅ Загрузка данных из Excel файла (листы "отпуска" и "дежурства") с поддержкой столбца "Отдел"
  • ✅ Фильтрация отображения по отделам с возможностью выбора нескольких отделов
  • ✅ Подробная подсказка при наведении на кнопку загрузки Excel (не выходит за пределы экрана)
  • ✅ Сообщения об успехе/ошибке загрузки (авто-скрытие через 5 сек)
